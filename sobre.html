Sobre o Projeto

Esse é um projeto que tem como objetivo a criação de um software que seja capaz de gerar imagens pela Inteligência Artificial. Essas imagens serão geradas a partir de um texto descritivo feito pelo usuário, onde ele irá explicar como quer que seja a imagem gerada. Por fim, as imagens geradas ficarão salvas na página "Galeria de Imagens".

Um pouco sobre as tecnologias que estamos utilizando:

  • Estamos utilizando o VS Code como IDE para suporte da nossa estrutura HTML.
  • Como dito anteriormente, também usaremos a estrutura HTML como nossa linguagem de marcação.
  • Por fim, usaremos o CSS e o JV para estilizar e aprimorar as páginas do nosso Projeto.
