Seja bem-vindo, esse Projeto consiste em um site gerador de imagens com IA a partir de um texto descritivo feito pelo usuário. Abaixo temos algumas das funcionaliades do site:

Funcionalidade Descrição
Texto Para o usuário descrever a imagem
Gerar Usado para criar a imagem de acordo com o texto
Galeria Armazena as imagens
